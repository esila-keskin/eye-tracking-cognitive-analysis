\# Eye Tracking for Cognitive Assessment



This project explores the use of eye-tracking data collected during game-based tasks to investigate relationships with age, cognitive status, and task performance.



\## Project goals

\- Clean and preprocess raw eye-tracking time-series data

\- Extract meaningful eye-movement features (e.g. fixations, saccades, temporal patterns)

\- Aggregate features at game and participant level

\- Explore baseline regression and classification models for cognitive and age-related measures



\## Structure

\- "data/" - raw and processed eye-tracking data (kept locally)

\- "notebooks/" - Jupyter notebooks for analysis and experimentation

\- "src/" - reusable analysis and preprocessing code



\## Status

Early-stage development. Current focus is on data inspection and signal preprocessing.



\## Tools

Python, pandas, numpy, scipy, scikit-learn, matplotlib, Jupyter Notebook
